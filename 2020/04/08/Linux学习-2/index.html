  Linux cd 命令可以说是Linux中最基本的命令语句，其他的命令语句要进行操作，都是建立在使用 cd 命令上的。所以，学习Linux 常用命令，首先就要学好 cd 命令的使用方法技巧。

命令格式

$ cd [目录名]

命令功能

切换当前目录至目标目录

上栗子

一：进入系统根目录

$ cd /

二：进入父级目录

$ cd ..
$ cd ..//

三：

$ cd
$ cd ~

四：跳转到指定目录

$ cd /usr/bin

五：返回进入此目录之前所在的目录

$ cd -

六：把上个命令的参数作为cd参数使用

$ cd !$
